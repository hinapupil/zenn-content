AIエージェントを無料で使いたい人向け。
Gemini APIは無料枠がかなり大きい[^1]。
Gemini CLI[^2]も無料枠が大きいが、使い勝手は微妙なので、ClineフォークのRooCodeにGemini APIを使用する。

## Gemini API 取得方法

1. https://ai.google.dev/gemini-api/docs/pricing?hl=ja
2. [Google AI Studio で試す] をクリック
3. [Get API key] をクリック
4. [API キーを作成] をクリック
5. [Gemini API] を選択（※出ないかも[^3]）
6. [既存のプロジェクトで API キーを作成] をクリック（※出ないかも[^3]）
7. APIキーを[コピー]
8. プランにFreeと書いてあることを確認（しばらく経たないと表示されないかも）
   ![](https://storage.googleapis.com/zenn-user-upload/4001a34230ad-20250715.png)

## VSCodeにRooCodeをインストール・設定

1. VSCodeを開いて、拡張機能ビューを表示する（サイドバーの四つの四角形のアイコンをクリック）

2. 検索バーに `RooVeterinaryInc.roo-cline` と入力し、表示された拡張機能をインストール

3. RooCodeを開いて、右上の歯車アイコンをクリック

4. [プロバイダー]タブで、以下を設定する

   | 設定項目        | 値                |
   | --------------- | ----------------- |
   | APIプロバイダー | Google Gemini     |
   | Gemini APIキー  | コピーしたAPIキー |
   | モデル          | gemini-2.5-pro OR gemini-2.5-flash |
   | 推論を有効化[^4] | ✓ |
   | レート制限      | 2.5-proなら6s, 2.5-flashなら12s[^5] |

   ※ 他はデフォルトでOK

6. RooCodeに全権任せて良い場合、[自動承認]タブで以下をすべてクリックして有効化する
   ![](https://storage.googleapis.com/zenn-user-upload/33fae3e2d1ed-20250715.png)

7. [自動承認]タブの[保護されたファイルを含める]を有効化する

8. [保存] をクリック

## RooCode の使い方

公式サイト: https://roocode.com/
公式ドキュメント: https://docs.roocode.com/

[自動承認]を有効にすると自動で読み書きやコマンド実行を行ってくれる。
モードは **Orchestrator** にするといい感じに自律的にモード切替とかしてやってくれる。
